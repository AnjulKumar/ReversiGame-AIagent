In this project, the program determines the next move for a player in the Reversi game using the Greedy, Minimax, and Alpha-Beta pruning algorithms with positional weight
evaluation functions.
------------------------------------------------
Input:
You are provided with a file input.txt that describes the current state of the game.
<task#> Greedy = 1, MiniMax = 2, Alpha-beta = 3, Competition =4
<your player: X or O>
<cutting off depth >
<current state as follows:>
*: blank cell
X: Black player
O: White Player

Output:
<next state>
<traverse log for minimax and alpha-beta pruning only>
